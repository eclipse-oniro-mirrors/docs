# 设备管理Changelog


## cl.device_manager.1 接口权限变更

从Opeharmonny 4.0.8.2版本开始，部分接口新增权限校验。

**变更影响**

在Opeharmonny 4.0.8.2之后版本开发的应用，需要持有相应权限，否则无法正常调用接口。
