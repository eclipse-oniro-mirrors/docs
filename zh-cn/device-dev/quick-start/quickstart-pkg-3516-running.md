# 运行


系统启动成功后，取源码out目录下的helloworld可执行文件放入系统的bin目录，通过以下步骤运行“Hello World”。


## 操作步骤

1. 在启动界面进入bin目录。
   
   ```
   cd bin
   ```

2. 进入bin目录后可以看到helloworld文件，通过以下命令运行helloworld程序。
   
   ```
   ./helloworld
   ```

   界面打印“Hello World！”，程序运行成功。
