# HdfMotionEvent


## 概述

上报手势识别数据结构。

上报手势识别数据事件信息包括手势识别类型、手势识别结果、手势识别状态、手势识别数据长度、手势识别数据。

**起始版本：** 3.2

**相关模块：**[Motion](_motion_v10.md)
