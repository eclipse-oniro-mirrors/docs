# PortCap


## **概述**

定义音视频编解码能力。

**相关模块:**

[Codec](codec.md)


## **汇总**


### Public 属性

  | 名称 | 描述 | 
| -------- | -------- |
| [video](#video) | 视频编解码能力 | 
| [audio](#audio) | 音频编解码能力 | 


## **类成员变量说明**


### audio

  
```
AudioPortCap PortCap::audio
```

**描述：**

音频编解码能力。


### video

  
```
VideoPortCap PortCap::video
```

**描述：**

视频编解码能力。
