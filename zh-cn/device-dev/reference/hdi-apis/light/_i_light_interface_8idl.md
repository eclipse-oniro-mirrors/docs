# ILightInterface.idl


## 概述

定义灯模块的通用接口能力，包括获取灯类型ID、打开或关闭灯光、设置灯的亮度和闪烁模式。

模块包路径：ohos.hdi.light.v1_0

引用：ohos.hdi.light.v1_0.LightTypes

**起始版本：** 3.1
