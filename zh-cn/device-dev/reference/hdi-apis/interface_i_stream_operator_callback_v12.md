# IStreamOperatorCallback


## 概述

定义Camera设备流回调操作。

对Camera设备执行流回调的抓捕，结束，错误捕获和帧捕获等操作。

**起始版本：** 4.1

**相关模块：**[Camera](_camera_v12.md)
