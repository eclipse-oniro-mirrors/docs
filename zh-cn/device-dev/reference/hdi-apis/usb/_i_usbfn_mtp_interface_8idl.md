# IUsbfnMtpInterface.idl


## 概述

声明usb模块提供的API，用于获取usb信息、订阅或取消订阅usb数据、启用或禁用usb、设置usb数据报告模式以及设置准确性和测量范围等usb选项。

模块包路径：ohos.hdi.usb.gadget.mtp.v1_0
