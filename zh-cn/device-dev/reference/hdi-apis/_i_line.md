# ILine


## **概述**

线条描述结构体定义，用于硬件加速绘制直线。

**相关模块:**

[Display](_display.md)

| [x1](_display.md#x1-13) | 线条终点的X坐标。 | 
| [y1](_display.md#y1-13) | 线条终点的Y坐标。 | 
| [color](_display.md#color-15) | 线条颜色。 |
