# VGUSurface


## **概述**

2D硬件加速绘制目标表面。

**相关模块:**

[Display](_display.md)


## **汇总**


### Public 属性

  | 名称 | 描述 | 
| -------- | -------- |
| [buffer](_display.md#buffer-23) | 位图缓存 | 
| [clipRect](_display.md#cliprect) | 矩形剪切域，如果为空，整个表面直接渲染。 | 
| [clipPath](_display.md#clippath) | 路径剪切域，如果为空，整个表面直接渲染。 | 
| [clipType](_display.md#cliptype) | 表面剪切类型 | 
| [mask](_display.md#mask) | 蒙版图层,&nbsp;可以为空。 | 
| [blend](_display.md#blend) | 混合叠加模式 | 
| [filter](_display.md#filter) | 图像滤波类型 |
