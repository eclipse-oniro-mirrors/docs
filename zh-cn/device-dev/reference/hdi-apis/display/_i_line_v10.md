# ILine


## 概述

线条描述结构体定义，用于硬件加速绘制直线。

**起始版本：** 3.2

**相关模块：**[Display](_display_v10.md)


## 汇总


### Public 属性

| 名称 | 描述 | 
| -------- | -------- |
| int [x0](#x0) | 线条起点的X坐标  | 
| int [y0](#y0) | 线条起点的Y坐标  | 
| int [x1](#x1) | 线条终点的X坐标  | 
| int [y1](#y1) | 线条终点的Y坐标  | 
| unsigned int [color](#color) | 线条颜色  |
