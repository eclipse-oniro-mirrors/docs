# LayerColor


## 概述

定义图层颜色设置的结构结构体。

**起始版本：** 3.2

**相关模块：**[Display](_display_v10.md)


## 汇总


### Public 属性

| 名称 | 描述 | 
| -------- | -------- |
| unsigned char [r](#r) | 图层色彩空间的红色值  | 
| unsigned char [g](#g) | 图层色彩空间的绿色值  | 
| unsigned char [b](#b) | 图层色彩空间的蓝色值  | 
| unsigned char [a](#a) | 图层色彩空间的不透明度  |
