# IRilCallback


## 概述

Ril模块的回调接口。

回调接口提供打电话、发短彩信、激活SIM卡、上网等回调函数，回调函数由调用者实现。

**起始版本：** 3.2

**相关模块：**[Ril](_ril_v11.md)
