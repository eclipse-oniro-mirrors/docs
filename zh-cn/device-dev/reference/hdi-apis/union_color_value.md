# ColorValue


## **概述**

定义灯的颜色模式。

参数包括RGB模式和WRGB模式。

**相关模块:**

[Light](light.md)


## **汇总**


### Public 属性

  | 名称 | 描述 | 
| -------- | -------- |
| [wrgbColor](#wrgbcolor) | WRGB模式 | 
| [rgbColor](#rgbcolor) | RGB模式 |
