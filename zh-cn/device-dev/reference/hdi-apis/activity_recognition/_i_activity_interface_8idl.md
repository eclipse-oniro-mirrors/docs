# IActivityInterface.idl


## 概述

声明行为识别模块提供的API，用于获取设备支持的行为类型，订阅或取消订阅不同的行为事件，获取当前的行为事件，以及获取设备缓存的行为事件。

模块包路径：ohos.hdi.activity_recognition.v1_0
