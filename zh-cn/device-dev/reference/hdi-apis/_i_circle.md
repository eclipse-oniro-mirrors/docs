# ICircle


## **概述**

圆形描述结构体定义，用于硬件加速绘制圆形。

**相关模块:**

[Display](_display.md)

| [r](_display.md#r) | 圆的半径。 | 
| [color](_display.md#color-25) | 圆的颜色。 |
