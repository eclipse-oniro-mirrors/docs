# InputTypes.idl


## 概述

Input设备相关的数据类型定义。

本模块为Input服务定义了设备驱动接口所使用的结构体类型，包括设备信息、设备属性、设备能力等。

模块包路径：ohos.hdi.input.v1_0

**起始版本：** 3.2

**相关模块：**[HdiInput](_hdi_input.md)
