# HdfHuks


## 概述

提供通用密钥库服务（OpenHarmony Universal KeyStore，简称HUKS）核心逻辑的标准驱动API接口。

HUKS驱动API定义了HUKS核心组件（HUKS Core）的标准接口，向HUKS Service提供了统一的驱动接口， 这些接口涵盖了密钥全生命周期管理，包括密钥生成、密钥导入导出、密钥操作、密钥访问控制、密钥证明等功能。

**起始版本：** 4.0
