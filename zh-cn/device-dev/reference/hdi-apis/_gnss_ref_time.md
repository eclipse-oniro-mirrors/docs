# GnssRefTime


## 概述

**起始版本：** 3.2

**相关模块：**[HdiGnss](_hdi_gnss.md)

### Public 属性

| 名称 | 描述 |

| long [timeMs](#timems) | UTC时间  | 
| long [timeReferenceMs](#timereferencems) | 自系统启动后经过的时间  | 
| int [uncertainty](#uncertainty) | 时间不确定度  |
