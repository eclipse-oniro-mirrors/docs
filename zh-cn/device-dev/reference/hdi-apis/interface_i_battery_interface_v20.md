## 概述

### ChangePath()

```
IBatteryInterface::ChangePath ([in] String path)
```

**描述**

设置电池信息节点的路径。

**起始版本：** 3.1

**参数:**

| 名称 | 描述 | 
| -------- | -------- |
| path | 输入参数，电池信息节点的路径。 | 

**返回：**

HDF_SUCCESS 表示注册成功。

HDF_FAILED 表示注册失败。
