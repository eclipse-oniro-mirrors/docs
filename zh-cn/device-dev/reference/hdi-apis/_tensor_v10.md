### quantParams

```
struct QuantParam [] Tensor::quantParams
```
**描述**
张量的量化参数数组。详情请参考：[QuantParam](_quant_param_v10.md)。

分为两种情况，如果长度为一，则所有轴公用一个量化； 若长度不为一，则数组中的每一个量化参数和轴一一对应。
