# AvgPoolFusion


## 概述

在输入tensor上应用 2D 平均池化。支持int8量化输入。

该算子对应的[NodeType](_n_n_rt_v20.md#nodetype)为NODE_TYPE_AVGPOOL_FUSION。

参数限制：当padMode==PAD_MODE_PAD时，padList的数值必须大于等于0。其他情况下padding的数值必须为0。

输入：

- x，n维张量。

输出：

- output， 输出平均池化后的张量。

**起始版本：** 3.2

**相关模块：**[NNRt](_n_n_rt_v20.md)
