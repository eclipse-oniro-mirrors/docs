# MaxPoolFusion

## 概述

对输入x，计算 2D 最大值池化。

该算子对应的[NodeType](_n_n_rt_v20.md#nodetype)为NODE_TYPE_MAX_POOL_FUSION。

参数限制：当padMode==PAD_MODE_PAD时，padList的数值必须大于等于0。其他情况下padding的数值必须为0。

输入：

- x，n维tensor。

输出：

- output，x1和x2两个tensor对应元素的最大值。

**起始版本：** 3.2

**相关模块：**[NNRt](_n_n_rt_v20.md)
