# DivFusion


## 概述

将两个tensor执行除法运算。

该算子对应的[NodeType](_n_n_rt_v10.md#nodetype)为NODE_TYPE_DIV_FUSION。

输入：

- x1，int或float类型的张量。

- x2，int或float类型的张量。

输出：

- output， 输出两输入相除后的结果。

**起始版本：** 3.2

**相关模块：**[NNRt](_n_n_rt_v10.md)
