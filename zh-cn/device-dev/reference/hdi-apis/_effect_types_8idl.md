# EffectTypes.idl


## 概述

Effect模块接口定义中使用的数据类型。

Effect模块接口定义中使用的控制器参数、控制器描述符、音效输入输出buffer参数、控制命令等。

模块包路径：ohos.hdi.audio.effect.v1_0

**起始版本：** 4.0

**相关模块：**[HdiEffect](_hdi_effect.md)
