# IntelligentVoiceTrigger


## 概述

IntelligentVoiceTrigger模块向上层服务提供了统一接口。

上层服务开发人员可根据IntelligentVoiceTrigger模块提供的向上统一接口获取如下能力：触发器适配器加载卸载、智能语音触发器模型加载卸载、底层唤醒业务启动停止等。

**起始版本：** 4.0
