# TimeEffect


## 概述

定义时间效果参数。

参数包括振动的延迟、时间、强度和频率。

**起始版本：** 3.2

**相关模块：**[Vibrator](_vibrator_v11.md)


## 汇总


### Public 属性

| 名称 | 描述 | 
| -------- | -------- |
| int [delay](#delay) | 等待时间  | 
| int [time](#time) | 振动时间  | 
| unsigned short [intensity](#intensity) | 振动强度  | 
| short [frequency](#frequency) | 振动频率（Hz）  |
