# QuantDTypeCast


## 概述

数据类型转换。

该算子对应的[NodeType](_n_n_rt_v10.md#nodetype)为NODE_TYPE_QUANT_DTYPE_CAST。

输入：

- x，n维tensor。

输出：

- output，类型转换之后的tensor。

**起始版本：** 3.2

**相关模块：**[NNRt](_n_n_rt_v10.md)


## 汇总
