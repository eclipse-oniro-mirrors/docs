# IGnssInterface.idl


## 概述

声明GNSS模块提供的接口函数，包括启动GNSS芯片、启动导航、设置GNSS工作模式、注入参考信息、 删除辅助数据、注入PGNSS数据、获取GNSS缓存位置个数、获取所有缓存位置。

模块包路径：ohos.hdi.location.gnss.v1_0

引用：

- ohos.hdi.location.gnss.v1_0.IGnssCallback

**起始版本：** 3.2
