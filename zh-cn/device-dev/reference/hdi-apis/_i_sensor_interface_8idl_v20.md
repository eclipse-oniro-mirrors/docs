# ISensorInterface.idl


## 概述

Sensor模块对外通用的接口声明文件，提供获取传感器设备信息、订阅/取消订阅传感器数据、 使能/去使能传感器、设置传感器模式、设置传感器精度，量程等可选配置接口定义。

模块包路径：ohos.hdi.sensor.v2_0

- ohos.hdi.sensor.v2_0.ISensorCallback
