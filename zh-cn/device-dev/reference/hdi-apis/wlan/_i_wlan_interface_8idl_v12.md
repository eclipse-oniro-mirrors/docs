# IWlanInterface.idl


## 概述

提供API以启用或禁用WLAN热点、扫描热点、连接到WLAN热点或断开与WLAN热点的连接，设置国家代码，并管理网络设备。

模块包路径：ohos.hdi.wlan.v1_2

引用：ohos.hdi.wlan.v1_1.IWlanInterface
