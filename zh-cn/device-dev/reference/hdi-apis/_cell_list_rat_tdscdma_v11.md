# CellListRatTdscdma


## 概述

TDSCDMA小区信息。

**起始版本：** 3.2

**相关模块：**[Ril](_ril_v11.md)


## 汇总


### Public 属性

| 名称 | 描述 | 
| -------- | -------- |
| int [arfcn](#arfcn) | BCCH载频的绝对射频频道号  | 
| int [syncId](#syncid) | 同步标识  | 
| int [sc](#sc) | 超级小区  | 
| int [cellId](#cellid) | 小区标识  | 
| int [lac](#lac) | 位置区号，取值范围0~0xFFFF  | 
| int [rscp](#rscp) | 接收信号码功率  | 
| int [drx](#drx) | 不连续接收周期长度  | 
| int [rac](#rac) | 路由区域码  | 
| int [cpid](#cpid) | 超级小区标识  |
