# WaveParam


## 概述

定义运动波配置参数。

运动波形配置参数包括波形频率、波形幅度、波形使用陀螺仪。

**起始版本：** 4.0

**相关模块：**[Motion](_motion_v11.md)


## 汇总


### Public 属性

| 名称 | 描述 | 
| -------- | -------- |
| int [waveFrequency](#wavefrequency) | 波浪频率  | 
| int [waveAmplitude](#waveamplitude) | 波幅  | 
| boolean [isUseGyroscope](#isusegyroscope) | 波动是使用陀螺仪  |
