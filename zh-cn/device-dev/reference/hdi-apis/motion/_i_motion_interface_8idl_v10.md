# IMotionInterface.idl


## 概述

定义使能/去使能手势识别、订阅/取消订阅手势识别数据的接口。

在实现拿起、翻转、摇一摇、旋转屏等手势识别功能时，需要调用此处定义的接口。

模块包路径：ohos.hdi.motion.v1_0

引用：

- ohos.hdi.motion.v1_0.MotionTypes

- ohos.hdi.motion.v1_0.IMotionCallback

**起始版本：** 3.2
