# IEffectModel


## 概述

音效模型接口。

提供音效模型支持的驱动能力，包括获取描述符列表、创建音效控制器、销毁音效控制器、获取指定描述符等

**起始版本：** 4.0

**相关模块：**[HdiEffect](_hdi_effect.md)

**描述**

**起始版本：** 4.0

**参数:**

| 名称 | 描述 | 
| -------- | -------- |
| model | 指向要操作的效果模型的指针 | 
| info | 音效信息 | 
| contoller | 音效控制器对象 |
