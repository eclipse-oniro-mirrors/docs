# CodecVideoParamHevc


## 概述

定义HEVC视频编码格式信息。

**起始版本：** 4.1

**相关模块：**[Codec](_codec_v20.md)

## 类成员变量说明


### keyFrameInterval

```
unsigned int CodecVideoParamHevc::keyFrameInterval
```
**描述**
连续I帧（包括其中一个I帧）之间的距离。 0表示间隔未指定，可由编解码器自由选择， 1表示一个仅存在I帧的流，其他值表示实际值。
