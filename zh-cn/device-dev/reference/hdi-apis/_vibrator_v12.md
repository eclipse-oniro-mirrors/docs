# Vibrator（V1_2）


## 概述

马达驱动对马达服务提供通用的接口能力。

模块提供马达服务对马达驱动访问的统一接口，服务获取驱动对象或者代理后，控制马达的单次振动、周期性振动、高清振动、停止振动、设置马达振幅与频率。

**起始版本：** 4.1

### EVENT_TYPE

```

```

**描述**

表示振动类型。

**起始版本：** 4.1

| 枚举值 | 描述 | 
| -------- | -------- |
| CONTINUOUS | 表示振动是连续的。 | 
| TRANSIENT | 表示振动是瞬时的。 | 


### HdfVibratorModeV1_2


**描述**

表示StopV1_2接口可传入参数枚举。

**起始版本：** 4.1

| 枚举值 | 描述 | 
| -------- | -------- |
| HDF_VIBRATOR_MODE_ONCE | 表示给定持续时间内的单次振动。 |

| HDF_VIBRATOR_MODE_HDHAPTIC | 表示高清振动。 | 
| HDF_VIBRATOR_MODE_BUTT | 表示效果模式无效。 |
