# WRGBColor


## 概述

定义灯的WRGB模式。

**起始版本：** 3.2

**相关模块：**[Light](_light.md)


## 汇总


### Public 属性

| 名称 | 描述 | 
| -------- | -------- |
| int [w](#w) | 白色值，范围为0-255。  | 
| int [r](#r) | 红色值，范围为0-255。  | 
| int [g](#g) | 绿色值，范围为0-255。  | 
| int [b](#b) | 蓝色值，范围为0-255。  |
