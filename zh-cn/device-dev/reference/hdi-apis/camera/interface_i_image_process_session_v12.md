# IImageProcessSession

## 概述

图像处理会话进程。

获取Coucurrency、待处理图像，设置执行模式，处理流程图像，删除图像，执行会话中断，会话重启。

**起始版本：** 4.1

**相关模块：**[Camera](_camera_v12.md)
