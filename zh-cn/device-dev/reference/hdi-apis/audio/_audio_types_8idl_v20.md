# AudioTypes.idl


## 概述

Audio模块接口定义中使用的数据类型。

Audio模块接口定义中使用的数据类型，包括音频端口、适配器描述符、设备描述符、场景描述符、采样属性、时间戳等。

模块包路径：ohos.hdi.audio.v2_0

**起始版本：** 4.1

**相关模块：**[HdiAudio](_hdi_audio_v20.md)
