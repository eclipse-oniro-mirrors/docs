# IPowerInterface


## 概述

休眠/唤醒操作、订阅休眠/唤醒状态、运行锁管理的接口。

**起始版本：** 3.1

**相关模块：**[Power](power_v11.md)

| [StartSuspend](#startsuspend) () | 执行设备休眠操作。  |
