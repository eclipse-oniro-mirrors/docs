# HostapdTypes.idl


## 概述

定义与WLAN模块相关的数据类型。

WLAN模块数据包括Feature对象信息、站点(STA)信息等，扫描信息和网络设备信息。

模块包路径：ohos.hdi.wlan.hostapd.v1_0

**起始版本：** 4.1

**相关模块：**[Hostapd](_hostapd.md)
