### ParamCallback()

```
IAudioCallback::ParamCallback ([in] enum AudioExtParamKey key, [in] String condition, [in] String value, [out] byte reserved, [in] byte cookie )
```
**描述**

**起始版本：** 4.1

**参数:**

| 名称 | 描述 | 
| -------- | -------- |
| key | 扩展参数键类型，详请参考[AudioExtParamKey](_hdi_audio_v20.md#audioextparamkey)。  |
