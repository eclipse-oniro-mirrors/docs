# Thermal（V1_1）


## 概述

提供设备温度管理、控制及订阅接口。

热模块为热服务提供的设备温度管理、控制及订阅接口。 服务获取此模块的对象或代理后，可以调用相关的接口管理、控制和订阅设备温度。

**起始版本：** 4.0
