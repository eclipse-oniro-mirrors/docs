# CryptoInfo


## 概述

定义CryptoInfo，该信息由[IMediaDecryptModule::DecryptMediaData](interface_i_media_decrypt_module.md#decryptmediadata)使用。

**起始版本：** 4.1

**相关模块：**[HdiDrm](_hdi_drm.md)
