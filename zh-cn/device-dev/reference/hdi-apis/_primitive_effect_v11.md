# PrimitiveEffect


## 概述

定义基本效果参数。

参数包括延迟、效应id和振动强度。

**起始版本：** 3.2

**相关模块：**[Vibrator](_vibrator_v11.md)


## 汇总


### Public 属性

| 名称 | 描述 | 
| -------- | -------- |
| int [delay](#delay) | 等待时间  | 
| int [effectId](#effectid) | 效果id  | 
| unsigned short [intensity](#intensity) | 振动强度  |
