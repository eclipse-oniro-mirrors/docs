# PowerTypes.idl


## 概述

电源相关的数据类型。

电源管理中使用的数据类型，包括命令参数、回调参数和系统状态。

模块包路径：ohos.hdi.power.v1_1

**起始版本：** 3.1

**相关模块：**[Power](power_v11.md)
