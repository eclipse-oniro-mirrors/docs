# ChargingLimit


## 概述

定义电池充电电流或电压的限制。

**起始版本：** 3.2

**相关模块：**[Battery](battery_v12.md)


## 汇总


### Public 属性

| 名称 | 描述 | 
| -------- | -------- |
| enum [ChargingLimitType](battery_v12.md#charginglimittype)[type](#type) | 定义电池充电限制类型  | 
| String [protocol](#protocol) | 限制协议描述  | 
| int [value](#value) | 选择限制类型，0-电流，1-电压  |
