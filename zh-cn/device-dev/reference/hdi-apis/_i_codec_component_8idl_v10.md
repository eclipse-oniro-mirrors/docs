# ICodecComponent.idl


## 概述

主要包括Codec组件接口定义。

Codec模块提供了获取组件信息、给组件发送命令、组件参数设置、buffer轮转和控制等接口定义。创建组件后，可使用下列接口进行编解码处理。

模块包路径：ohos.hdi.codec.v1_0

引用：

- ohos.hdi.codec.v1_0.CodecTypes

- ohos.hdi.codec.v1_0.ICodecCallback

**起始版本：** 3.2

**相关模块：**[Codec](_codec_v10.md)
