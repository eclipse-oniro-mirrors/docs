# INfcInterface


## 概述

声明操作nfc芯片的API，包括关闭、打开nfc，初始化nfc，读写数据、配置RF参数、发送nci指令。

**起始版本：** 3.2

**相关模块：**[HdiNfc](_hdi_nfc_v10.md)
