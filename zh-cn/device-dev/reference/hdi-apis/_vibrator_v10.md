# Vibrator（V1_0）


## 概述

马达驱动对马达服务提供通用的接口能力。

模块提供马达服务对马达驱动访问的统一接口，服务获取驱动对象或者代理后，控制马达的单次振动、周期性振动、停止振动。

**起始版本：** 2.2

### HdfVibratorMode

```

```

**描述**

定义马达振动模式。

**起始版本：** 2.2

| 枚举值 | 描述 | 
| -------- | -------- |
| HDF_VIBRATOR_MODE_ONCE | 表示给定持续时间内的单次振动。 | 
| HDF_VIBRATOR_MODE_PRESET | 表示具有预置效果的周期性振动。 | 
| HDF_VIBRATOR_MODE_BUTT | 表示效果模式无效。 |
