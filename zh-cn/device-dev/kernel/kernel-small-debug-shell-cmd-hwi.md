# hwi


## 命令功能

hwi命令查询当前中断信息


## 命令格式

hwi


## 参数说明

无


## 使用指南

- 输入hwi即显示当前中断号、中断次数及注册中断名称。

- 若开关LOSCFG_CPUP_INCLUDE_IRQ打开，则还会显示各个中断的处理时间ATime、CPU占用率以及中断类型。


## 使用实例

举例：输入hwi
