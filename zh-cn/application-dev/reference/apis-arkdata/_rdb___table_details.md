# Rdb_TableDetails


## 概述

描述数据库表执行端云同步任务上传和下载的统计信息。

**起始版本：** 11

**相关模块：**[RDB](_r_d_b.md)


## 汇总


### 成员变量

| 名称 | 描述 |
| -------- | -------- |
| [table](_r_d_b.md#table) | 数据库表名。 |
| [upload](_r_d_b.md#upload) | 表示数据库表中端云同步上传过程的统计信息。 |
| [download](_r_d_b.md#download) | 表示数据库表中端云同步下载过程的统计信息。 |
