# AbilityBase

## 概述

作为Ability Kit的基础定义模块，AbilityBase提供了组件启动参数[Want](want__8h.md)的定义与接口，可以用于应用组件间的信息传递。

**系统能力**：SystemCapability.Ability.AbilityBase

**起始版本**：15
