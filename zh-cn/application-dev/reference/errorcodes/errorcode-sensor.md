# 传感器错误码

> **说明：**
>
> 以下仅介绍本模块特有错误码，通用错误码请参考[通用错误码说明文档](errorcode-universal.md)。

## 14500101 传感器服务异常

**错误信息**

Service exception.

**错误描述**

当调用sensor模块on、once、off接口时，若hdi服务异常，会报此错误码。

**可能原因**

访问hdi服务状态异常。

**处理步骤**

1. 定时重试操作，如间隔1s或者按照指数增长间隔重试。
2. 连续重试3次不可用则停止尝试，期间可优先尝试获取器件列表方式进一步获取设备可用性。
