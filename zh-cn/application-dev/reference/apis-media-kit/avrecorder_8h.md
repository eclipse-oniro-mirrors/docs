# avrecorder.h


## 概述

定义AVRecorder接口。应用可使用Media AVRecorder提供的接口录制媒体数据。

**库：** libavrecorder.so

**引用文件：** &lt;multimedia/player_framework/avrecorder.h&gt;

**系统能力：** SystemCapability.Multimedia.Media.AVRecorder

**起始版本：** 18

**相关模块：**[AVRecorder](_a_v_recorder.md)
