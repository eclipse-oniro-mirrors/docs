# avrecorder_base.h


## 概述

定义了媒体 AVRecorder 的结构体和枚举。

**库：** libavrecorder.so

**引用文件：** &lt;multimedia/player_framework/avrecorder_base.h&gt;

**起始版本：** 18

**相关模块：**[AVRecorder](_a_v_recorder.md)
