## WebMessage

type WebMessage = ArrayBuffer | string

用于描述[WebMessagePort](./js-apis-webview-WebMessagePort.md)所支持的数据类型。

**系统能力：** SystemCapability.Web.Webview.Core

| 类型       | 说明                                     |
| -------- | -------------------------------------- |
| string   | 字符串类型数据。 |
| ArrayBuffer   | 二进制类型数据。 |
