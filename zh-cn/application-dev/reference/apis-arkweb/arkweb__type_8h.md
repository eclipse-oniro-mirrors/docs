# arkweb_type.h


## 概述

提供ArkWeb在Native侧的公共类型定义。

**库：** libohweb.so

**引用文件：** <web/arkweb_type.h>

**系统能力：** SystemCapability.Web.Webview.Core

**起始版本：** 12

**相关模块：**[Web](_web.md)
