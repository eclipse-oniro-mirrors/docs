# arkweb_scheme_handler.h


## 概述

声明用于拦截来自ArkWeb的请求的API。

**库：** libohweb.so

**引用文件：** <web/arkweb_scheme_handler.h>

**系统能力：** SystemCapability.Web.Webview.Core

**起始版本：** 12

**相关模块：**[Web](_web.md)
