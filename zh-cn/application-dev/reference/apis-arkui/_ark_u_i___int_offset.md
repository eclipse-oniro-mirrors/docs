# ArkUI_IntOffset


## 概述

位置，用于描述组件的位置。

**起始版本：** 12。

**相关模块：**[ArkUI_NativeModule](_ark_u_i___native_module.md)


## 汇总


### 成员变量

| 名称 | 描述 | 
| -------- | -------- |
| int32_t [x](#x) | 水平方向横坐标，单位为px。  | 
| int32_t [y](#y) | 竖直方向纵坐标，单位为px。  | 


## 结构体成员变量说明


### x

```
int32_t ArkUI_IntOffset::x
```
**描述：**

水平方向横坐标，单位为px。


### y

```
int32_t ArkUI_IntOffset::y
```
**描述：**
