# ArkUI_IntSize


## 概述

尺寸类型，用于描述组件的宽高。

**起始版本：** 12。

**相关模块：**[ArkUI_NativeModule](_ark_u_i___native_module.md)

**所在头文件：** [native_type.h](native__type_8h.md)

## 汇总


### 成员变量

| 名称 | 描述 | 
| -------- | -------- |
| int32_t [width](#width) | 宽度，单位为px。  | 
| int32_t [height](#height) | 高度，单位为px。  | 


## 结构体成员变量说明


### height

```
int32_t ArkUI_IntSize::height
```
**描述：**

高度，单位为px。


### width

```
int32_t ArkUI_IntSize::width
```
**描述：**

宽度，单位为px。
