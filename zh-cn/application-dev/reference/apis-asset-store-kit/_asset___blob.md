# Asset_Blob


## 概述

二进制数组类型，即不定长的字节数组。

**起始版本：** 11

**相关模块：**[AssetType](_asset_type.md)

**所在头文件：**[asset_type.h](asset__type_8h.md)


## 汇总


### 成员变量

| 名称 | 描述 |
