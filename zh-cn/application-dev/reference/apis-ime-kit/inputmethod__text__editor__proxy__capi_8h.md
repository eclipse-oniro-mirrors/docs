# inputmethod_text_editor_proxy_capi.h


## 概述

提供一套方法支持应用开发的自绘输入框获取来自输入法应用的通知和请求。

**库：** libohinputmethod.so

**系统能力：** SystemCapability.MiscServices.InputMethodFramework

**起始版本：** 12

**相关模块：**[InputMethod](_input_method.md)
