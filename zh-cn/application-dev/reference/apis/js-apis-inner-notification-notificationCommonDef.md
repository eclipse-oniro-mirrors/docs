# NotificationCommonDef

> **说明：**
>
> 本模块首批接口从API version 9开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本。

## BundleOption

BundleOption模块为指定应用的包信息。

**系统能力**：以下各项对应的系统能力均为SystemCapability.Notification.Notification

| 名称   | 类型   | 只读 | 必填 | 说明   |
| ------ | ------ | ---- | ---- |  ------ |
| bundle | string | 否   | 是 | 应用的包信息。 |
| uid    | number | 否   | 否 | 用户ID。 |
