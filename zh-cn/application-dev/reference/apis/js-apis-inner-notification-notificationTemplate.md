# NotificationTemplate

通知模板。

> **说明：**
>
> 本模块首批接口从API version 8开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本。

**系统能力**：以下各项对应的系统能力均为SystemCapability.Notification.Notification

| 名称 | 类型                    | 只读 | 必填 | 说明       |
| ---- | ---------------------- | ---- | ---- | ---------- |
| name | string                 | 否   | 是   | 模板名称。 |
| data | {[key:string]: Object} | 否   | 是   | 模板数据。 |
