# drawing_text_line.h


## 概述

提供获取文本行内的字符位置、获取渲染单元信息和按行截断等功能。

**引用文件：** <native_drawing/drawing_text_line.h>

**库：** libnative_drawing.so

**起始版本：** 16

**相关模块：**[Drawing](_drawing.md)
