# drawing_text_run.h


## 概述

提供字体渲染单元的相关接口，比如绘制功能、获取排版边界功能等。 

**引用文件：**<native_drawing/drawing_text_run.h>

**库：** libnative_drawing.so

**起始版本：** 16

**相关模块：**[Drawing](_drawing.md)
