# OH_Drawing_BitmapFormat


## 概述

结构体用于描述位图像素的格式，包括颜色类型和透明度类型。

**起始版本：** 8

**相关模块：**[Drawing](_drawing.md)

**所在头文件：**[drawing_bitmap.h](drawing__bitmap_8h.md)
