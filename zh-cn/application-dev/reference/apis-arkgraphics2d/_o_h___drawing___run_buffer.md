# OH_Drawing_RunBuffer


## 概述

结构体用于描述一块内存，描述文字和位置信息。

**起始版本：** 11

**相关模块：**[Drawing](_drawing.md)


## 汇总


### 成员变量

| 名称 | 描述 |
| -------- | -------- |
| uint16_t \* [glyphs](#glyphs) | 存储文字索引。 |
| float \* [pos](#pos) | 存储文字的位置。 |
| char \* [utf8text](#utf8text) | 存储文字UTF-8编码。 |
| uint32_t \* [clusters](#clusters) | 存储文字簇UTF-8编码（簇指的是集合）。 |
