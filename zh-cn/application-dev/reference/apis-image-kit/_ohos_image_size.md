# OhosImageSize


## 概述

定义图像大小。

**系统能力：** SystemCapability.Multimedia.Image

**起始版本：** 10

**相关模块：** [Image](image.md)


## 汇总


### 成员变量

| 名称 | 描述 | 
| -------- | -------- |
| [width](#width) | 像素中的图像宽度，用pixels表示 | 
| [height](#height) | 像素中的图像高度，用pixels表示 |
