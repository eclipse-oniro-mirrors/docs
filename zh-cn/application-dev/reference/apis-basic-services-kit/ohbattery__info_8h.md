# ohbattery_info.h


## 概述

声明电池API以获取当前电池容量和电源类型的信息，定义电池相应常见事件。

**库：** libohbattery_info.so

**系统能力：** SystemCapability.PowerManager.BatteryManager.Core

**起始版本：** 13

**相关模块：**[OH_BatteryInfo](oh__batteryinfo.md)
