# oh_pasteboard.h


## 概述

提供访问系统剪贴板的接口、数据结构、枚举类型。 

**库：** libpasteboard.so

**引用文件：** <database/pasteboard/oh_pasteboard.h>

**系统能力：** SystemCapability.MiscServices.Pasteboard

**起始版本：** 13

**相关模块：**[Pasteboard](_pasteboard.md)
