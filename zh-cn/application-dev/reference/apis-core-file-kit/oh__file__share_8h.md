# oh_file_share.h


## 概述

提供基于URI的文件及目录授于持久化权限、权限激活、权限查询等方法。

**库：** libohfileshare.so

**系统能力：** SystemCapability.FileManagement.AppFileService.FolderAuthorization

**起始版本：** 12

**相关模块：**[FileShare](file_share.md)
