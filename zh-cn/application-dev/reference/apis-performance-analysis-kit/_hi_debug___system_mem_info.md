# HiDebug_SystemMemInfo

**起始版本：** 12

**相关模块：** [HiDebug](_hi_debug.md)

**所在头文件：** [hidebug_type.h](hidebug__type_8h.md)

### 成员变量

| 名称 | 描述 |

| uint32_t [totalMem](#totalmem) | 系统总的内存，以KB为单位。  |
