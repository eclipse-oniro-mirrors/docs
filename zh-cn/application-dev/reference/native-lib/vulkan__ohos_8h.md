# vulkan_ohos.h


## 概述

定义了OpenHarmony平台扩展的Vulkan接口。

**引用文件：** &lt;vulkan/vulkan.h&gt;。

**起始版本：**

10

**相关模块：**

[Vulkan](_vulkan.md)
