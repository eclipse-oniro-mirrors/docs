# OH_NativeImage


## 概述

提供NativeImage功能，作为数据消费者，主要用来将数据和OpenGL纹理对接，需在OpenGL环境下使用

\@syscap SystemCapability.Graphic.Graphic2D.NativeImage

**起始版本:**

9

### OH_NativeImage_AttachContext()

**参数:**

| 名称 | 描述 |
| -------- | -------- |
| image | 指向OH_NativeImage实例的指针。 |

**返回:**
