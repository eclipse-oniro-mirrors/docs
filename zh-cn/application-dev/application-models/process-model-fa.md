# 进程模型概述


系统的进程模型如下图所示：


- 应用中（同一包名）的所有PageAbility、ServiceAbility、DataAbility、FormAbility运行在同一个独立进程中，即图中绿色部分的“Main Process”。

- WebView拥有独立的渲染进程，即图中黄色部分的“Render Process”。
