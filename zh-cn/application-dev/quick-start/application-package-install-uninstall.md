# 应用程序包安装和卸载流程

## 开发者
开发者可以通过调试命令进行应用的安装和卸载，可参考[多HAP的调试流程](multi-hap-release-deployment.md#调试)。

**图1** 应用程序包安装和卸载流程（开发者）

## 终端设备用户
开发者将应用上架应用市场后，终端设备用户可以在终端设备上使用应用市场进行应用的安装和卸载。
