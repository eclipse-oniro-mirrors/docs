**Since**: 12

**Parameters**

| Name| Description| 
| -------- | -------- |
| algoName | Pointer to the algorithm used to create a symmetric cipher instance. For example, **'AES128\|GCM\|PKCS7'**.| | | 
| ctx | Pointer to the symmetric cipher instance created.|
