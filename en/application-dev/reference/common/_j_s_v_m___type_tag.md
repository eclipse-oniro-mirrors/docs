# JSVM_TypeTag


## Overview

Type token, stored as a 128-bit value of two unsigned 64-bit integers. As a UUID, JavaScript objects can be "tagged" through it to ensure that their types remain unchanged.

**Since**: 11

Related module: [JSVM](_j_s_v_m.md)
