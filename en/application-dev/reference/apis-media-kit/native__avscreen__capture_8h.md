# native_avscreen_capture.h


## Overview

The **native_avscreen_capture.h** file declares the native APIs used to create an **OH_AVScreenCapture** instance.

**Since**

10

**Related Modules**

[AVScreenCapture](_a_v_screen_capture.md)
