# native_interface_accessibility.h


## Overview

Declares APIs that enable third-party applications to access the native accessibility features.

**Library**: libace_ndk.z.so

**File to include**: <arkui/native_interface_accessibility.h>

**Since**: 8

**Related module**: [Native Accessibility](arkui_native_interface_accessibility.md)
